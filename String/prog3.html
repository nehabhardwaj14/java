3.Program to prove String is immutable programmatically?

            class Main{
                public static void main(String[] args){
                    String str = "Hello";
                    String strN = str + " World";
                    if(str == strN){
                        System.out.println("Both object are same i.e Mutable Objects");
                    }
                    else{
                        System.out.println("Both object are different i.e Immutable Objects");
                    }
                }
            }

OUTPUT

            Both object are different i.e Immutable Objects
