10. Program to check whether a string is a Palindrome

            class Main{
                static boolean isPallindrome(String str){
                    for(int i = 0, j = (str.length()-1); i < j; i++, j--){
                        if(str.charAt(i) != str.charAt(j)){
                            return false;
                        }
                    }
                    return true;
                }
                public static void main(String[] args){
                    System.out.println((isPallindrome("abc11cba")) ? "abc11cba is pallindrome" : "abc11cba is not pallindrome");
                    System.out.println((isPallindrome("call")) ? "39d is pallindrome" : "39d is not pallindrome");
                }
            }

OUTPUT

            abc11cba is pallindrome
            call is not pallindrome
