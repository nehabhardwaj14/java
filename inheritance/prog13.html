13. Create a program, showing an example of method overriding

            class Animal
            {
                void Lion()
            {
              System.out.println("Lion is the king of jungle");
            }
            }
            class Food extends Animal
            {
               void Lion()
            {
               System.out.println("Lion eats meet");
            }
            }
            class main
            {
               public static void main(String st[])
                 {
                    Food obj=new Food();
                            obj.Lion();
                 }
            }

OUTPUT

            Lion eats meet
