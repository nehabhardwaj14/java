12. Create a program, showing an example of super function

            class Playground
            {
               void play()
                {
                    System.out.println("children are playing");
                }
            }
            class Children extends Playground
            {
             void play()
            {
               super.play();
            }
            }
            class main
            {
             public static void main(String st[])
               {
                   Children obj=new Children();
                         obj.play();
            }
            }

OUTPUT

            children are playing
