INHERITANCE

5. Write a Java program to create an abstract class Animal with an abstract method called sound(). Create subclasses Lion and Tiger that extend the Animal class and implement the sound() method to make a specific sound for each animal.

            abstract class Animal
            {
              abstract void sound();
            }
            class Lion extends Animal
            {
               void sound()
                {
                   System.out.println("Lion roars");
               }
            }
            class Tiger extends Animal
            {
               void sound()
                {
                   System.out.println("Tiger growls");
               }
            }
            class main
            {
              public static void main(String st[])
                {
                   Animal obj=new Lion();
                    Animal obj1=new Tiger();
                           obj.sound();
                          obj1.sound();
               }
            }

OUTPUT

          Lion roars
          Tiger growls
