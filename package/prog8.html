8.Create a Java program to create a package with public class and protected members to be accessed in another class.

            package mypac1;
            public class First
            {
            protected int x=10;
            protected void printMessage()
            {
            System.out.println("hello first class");
            }
            }
          
        
            //in another file
            package mypac1;
            public class Second extends First
            {
        
                protected void accessProtected()
            {
              System.out.println("x= " + x);
               printMessage();
              }
            
            }
              import mypackage.Second;
        
              class main
              {
              public static void main(String st[])
              {
              Second obj=new Second();
              obj.accessProtected();
              }
              }

OUTPUT

            x=10
            hello first class
