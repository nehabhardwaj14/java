9.Create a Java program to create a nested package i.e. c4.sample with public class and protected members to be accessed in another class.

            package c4.sample;

            public class Myclas
            {
            protected int var1=10;
            protected void method()
            {
            System.out.println("protected method called");
            }
            }
        
            package c4.sample;
        public class Aclass extends Myclas
        {
        public static void main(String st[])
        {
         Aclass obj= new Aclass();
        System.out.println("protected var1:" + obj.var1);
        obj.method();
        }
        }
        
        import c4.sample.Myclas;
        
        public class outside extends Myclas
        {
        public static void main(String st[])
        {
        outside obj= new outside();
        System.out.println("protected var1:" + obj.var1);
        obj.method();
        }
        }

OUTPUT

            protected var1=10
            protected method called
