3. How to convert String to long

            class ConvertLong
            {
               public static void main(String st[])
                {
                   String numString="876756453";
                   long number = Long.parseLong(numString);
                   System.out.println("Converted String to long : " + number);
            }
            }

OUTPUT

            Converted String to long : 876756453
