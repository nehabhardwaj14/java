39. Java program to find surface area of cylinder

            class SurfaceAreaofCylinder
            {
               public static void main(String st[])
                 {
                          double radius=4.7;
                           double height=3.7;
            
                      double surfaceArea=2 * Math.PI * radius * (height + radius);
            
             System.out.println("surface area of cylinder:"+surfaceArea);
            }
            }

OUTPUT

            surface area of cylinder:248.06015592745007
