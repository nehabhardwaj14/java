26. Write a Java program to add all the digits of a given positive integer

            class AddDigit
            {
            public static void main(String st[])
                {
                int arr[]={1,2,3,4,5};
                int sum=0;
                for(int i=0;i<arr.length;i++)
                {
                    sum+=arr[i];
                }
                System.out.println("Add of digit : "+sum);
                }
            }

OUTPUT

            Add of digit : 15
