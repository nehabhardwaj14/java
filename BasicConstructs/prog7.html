7. Write a Java program to convert a decimal number to binary numbers

            class Main{
                public static void main(String[] args){
                    int num = 9, j = 0;
                    short arr[] = new short[10];
                    while(num > 0){
                        arr[j] = (short)(num % 2);
                        num /= 2;
                        j++;
                    }
                    for(int i = arr.length-1; i >= 0; i--){
                        System.out.print(arr[i]);
                    }		 
                }
            }

OUTPUT

            0000001001
