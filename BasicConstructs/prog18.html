18. Write a Java program to convert a string to an integer

            import java.util.Scanner;
  
            class ConvertString
             {
                 public static void main(String st[])
                  {
                         Scanner str=new Scanner(System.in);
                            System.out.print("Enter a string representing an integer: ");
                        String str1 = str.nextLine();
                     try {
                                int num = Integer.parseInt(str1);
                            System.out.println("The integer value is: " + num);
                          } 
                   catch (NumberFormatException e) {
                        System.out.println("Invalid input. Please enter a valid integer string.");
                    }
            }
            }

OUTPUT

            Enter a string representing an integer: 7895
            The integer value is: 7895
