4. Write a Java program to print the sum, multiply, subtract, divide and remainder of two numbers

            class Main
            {
               public static void main(String st[])
                {
                             int a=20;
                             int b=12;
                         int sum=a+b;
                         int subtract=a-b;
                         int product=a*b;
                         int divide=a/b;
                        int remainder=a%b;
                 System.out.println("Sum of Two number: "+sum);
                 System.out.println("Subtraction of Two number: "+subtract);
                 System.out.println("Product of Two number: "+product);
                 System.out.println("Divide of Two number: "+divide);
                 System.out.println("Remainder of Two number: "+remainder);
              }
            }

OUTPUT

            Sum of Two number: 32
            Subtraction of Two number: 8
            Product of Two number: 240
            Divide of Two number: 1
            Remainder of Two number: 8
