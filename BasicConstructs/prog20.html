20. Write a Java program to compute the sum of the first 100 prime numbers

            class SumofPrime
            {
                static Boolean computePrime(int num)
                   {
                        if(num<=1)
                         {
                             return false;
                         }
                    for(int i=2;i<=Math.sqrt(num); i++)
                     {
                          if(num%i==0)
                            {
                               return false;
                            }
                         }
                     return true;
                    }
            public static void main(String st[])
               {
                   int count=0;
                   int sum=0;
                    int num=2;
                 
                 while(count<100)
                  {
                      if(computePrime( num))
                         {
                            sum+=num;
                            count++;
                          }
                         num++;
                    }
                System.out.println("The sum of the first 100 prime numbers is: " + sum);
                }
            }

OUTPUT

            The sum of the first 100 prime numbers is: 24133
