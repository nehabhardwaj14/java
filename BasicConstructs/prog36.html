36. Java Program to find volume of cuboid

            class Cuboid
            {
               public static void main(String st[])
                 {
                          double length=4.7;
                           double width=5.4;
                           double height=3.7;
            
                      double volume= length * width * height;
            
             System.out.println("Volume of Cuboid:"+volume);
            }
            }

OUTPUT

            Volume of Cuboid:93.90600000000002
