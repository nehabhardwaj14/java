14. Write a Java program that accepts an integer (n) and computes the value of n+nn+nnn

            import java.util.Scanner;

            class ComputeValue
             {
                public static void main(String st[])
                 {
                    Scanner str=new Scanner(System.in);
                    System.out.print("Enter an integer (n): ");
                       int n = str.nextInt();
             
                    int nn = Integer.parseInt(String.valueOf(n) + n);
                    int nnn = Integer.parseInt(String.valueOf(n) + n + n);
            
                    int result = n + nn + nnn;
                    System.out.print("Value of n+nn+nnn is: "+result);
                 }
            }

OUTPUT

            Enter an integer (n): 6
            Value of n+nn+nnn is: 738
