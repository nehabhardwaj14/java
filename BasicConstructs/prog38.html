38. Java program to find surface area of cuboid

            class SurfaceAreaCuboid
            {
               public static void main(String st[])
                 {
                          double length=4.7;
                           double width=6.7;
                           double height=3.7;
            
                      double surfaceArea=2 * (length * width + length * height + width * height);
            
             System.out.println(" SurfaceArea of Cuboid:"+surfaceArea);
            }
            }

OUTPUT

            SurfaceArea of Cuboid:147.34
