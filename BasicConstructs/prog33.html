33. Java Program to find area of Prism

            class Prism
            {
               public static void main(String st[])
                 {
                     double base=10.6;
                     double perimeter = 18.0; 
                    double height = 10.0; 
            
                 double surfaceArea=2*base+perimeter*height ;
            System.out.println("Area of prism:"+surfaceArea);
            }
            }

OUTPUT

            Area of prism:201.2
