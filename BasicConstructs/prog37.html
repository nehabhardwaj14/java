37. Java Program to find volume of cone

            class Cone
            {
               public static void main(String st[])
                 {
                          double radius=4.7;
                           double height=3.7;
            
                      double volume=(1.0 / 3.0) * Math.PI * Math.pow(radius, 2) * height;
            
             System.out.println("Volume of Cone:"+volume);
            }
            }

OUTPUT

            Volume of Cone:85.59059745195154
