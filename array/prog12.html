12. Program to print the sum of all the items of the array

            class  SumCalculator
            {
                static int calculateSum(int arr[])
                  {
                             int sum=0;
                              for(int num:arr){
                                sum+=num;
                              }
                            return sum;
                    }
              public static void main(String st[])
                 {
                       int arr1[]={10,34,56,78,90};
                        System.out.println("Sum of Array elements:"+calculateSum(arr1));
                  }
            }

OUTPUT

            Sum of Array elements:268
