7. Program to print the elements of an array present on even position

            class EvenArrayElements
            {
               static void evenElements(int arr[])
                {
                 System.out.println("Elements present at even position are:");
                     for(int i=0; i<arr.length;i++){
                        if(i%2==0){
                          System.out.println(arr[i]);
                         }
                       }
                  }
              public static void main(String st[])
               {
                     int num[]={10,20,30,40,50};
                       evenElements(num);
               }
            }

OUTPUT

            Elements present at even position are:
            10
            30
            50
