8. Program to print the elements of an array present on odd position

            class OddArrayElements
            {
               static void oddElements(int arr[])
                {
                 System.out.println("Elements present at odd position are:");
                     for(int i=0; i<arr.length;i++){
                        if(i%2!=0){
                          System.out.println(arr[i]);
                         }
                       }
                  }
              public static void main(String st[])
               {
                     int num[]={10,20,30,40,50};
                       oddElements(num);
               }
            }

OUTPUT

            Elements present at odd position are:
            20
            40
